# Decodificador JS - Alura Challenges
## Sobre
 É o primeiro projeto realizado durante o curso da Oracle Next Education ONE e em parceria com a Alura. O projeto tem a função de decodificar e criptografar mensagens em um estilo próprio do curso.

Ele segue o simples padrão:
- A letra "e" é convertida para "enter"
- A letra "i" é convertida para "imes"
- A letra "a" é convertida para "ai"
- A letra "o" é convertida para "ober"
- A letra "u" é convertida para "ufat"

Por exemplo:
- "gato" => "gaitober"
- gaitober" => "gato"
